Intro - top

Angushop - Angular 4 Shop Template Material Design is a simple html eccommerce template for angular 4, and Material design with unique style. This template is very easy to customize with great structure code.

This theme is built upon Material Angular framework and is fully responsive. It adapts to window resolution and looks good on all devices including ipad and iphone. See for yourself (resize your browser). If you would like to read more about Material Angular 2 click here : Material Angular 2 Documentation

Features :
  1. Angular 4 / CLI
  2. Strip JS Checkout
  3. Fully Responsive
  4. Support modern browser
  5. 11 color options
  6. 5 Product Page
  7. 2 Home Page
  8. Material Framework
  9. Sass Processor css
  10. And many more
Intalation - top
  • Istall Node Js & angular CLI Click Here
  • open "terminal / cmd / git bash"
  • go to angushop path "cd ANGUSHOP-PATH"
  • npm install
  • after install package "ng serve" to run application

Go to documentation angular-cli

Select Theme - top

All theme located in "/assets/theme/css/" you can chose it with your favorite color, or you can custom it with scss processor in "/assets/theme/scss/"

Logo Setting - top

The current logo image located in folder "/assets/images/brand/logo.png" you can change it with your own logo by replacing current logo, make sure use same fine name and format ( logo.png ) so you don't need to edit the style.css file. logo size is 77px x 77px.

Favicon Setting - top

The favicon image located in folder "src/assets/favicon.png" you can change it with your favicon by replacing it, make sure use same fine name and format ( favicon.png ) the size is 300px x 300px.

Product Setting - top

import { libModule } from '../lib/lib.module';
...
@NgModule({
    imports: [
        ...
        libModule
        ...
    ],
...


<app-product 
    [filter]="{
        productName: 'Product name',          filter "regex" product name
        category: 'category product',         filter "equal" product category
        color: 'red',                         filter "equal" product color
        price: 3000,                          filter "set max" product price
        size: [33,34]                         filter "in" product size
        index: [11,12]                         filter "in" product index
    }"
    [column]="'l4 s6'" View grid
    [setlimit]="12" 
    [paginate]="true" 
    [showBtnClose]="false" 
    [currentPage]="1"
    (pageChange)="onPageChange($event)">
</app-product>
